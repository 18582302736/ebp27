如何获取 GitHub Token？
  1. 打开 github.com/settings/tokens（需登录 GitHub）
  2. 点击 "Generate new token" → "Generate new token (classic)"
  3. Note 填写 "ebp27-backup"，Expiration 选择 "No expiration"
  4. 勾选 "repo" 权限
  5. 点击底部 "Generate token"
  6. 复制生成的 token（以 ghp_ 开头）
  7. 粘贴到上方输入框，点击保存
